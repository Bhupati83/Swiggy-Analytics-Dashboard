# 🍊 Swiggy Bangalore Analytics Dashboard

## 📌 Project Overview
An end-to-end Data Analytics & Business Intelligence project 
analyzing 1,499 Swiggy restaurants across Bangalore using 
Python, Excel, and Power BI.

## 🎯 Problem Statement
Swiggy wants to understand restaurant performance across 
Bangalore to help businesses make data-driven decisions 
about location, cuisine, and pricing strategy.

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Python (Pandas, Matplotlib, Seaborn) | Data Cleaning & EDA |
| Excel | Data Summary & Pivot Reports |
| Power BI | Interactive Dashboard |

## 📊 Key Insights
- 55% restaurants are Mid-Range (₹200-400)
- Average rating across Bangalore: 3.67 ⭐
- Top area by restaurants: Koramangala (109 restaurants)
- Most popular cuisine: North Indian
- Average cost for two: ₹306
- Premium restaurants have higher ratings than Budget

## 🚀 How to Run
1. Clone this repository
2. Install dependencies:
   pip install pandas matplotlib seaborn openpyxl
3. Run Python files in order:
   python 01_data_cleaning.py
   python 02_eda_analysis.py
   python 03_export_to_excel.py
4. Open Swiggy_Dashboard.pbix in Power BI Desktop

## 📈 Dashboard Features
- KPI Cards — Total Restaurants, Avg Rating, Avg Cost
- Bar Chart — Top 10 Areas by Restaurant Count
- Pie Chart — Price Segment Distribution
- Bar Chart — Top 10 Cuisines
- Scatter Chart — Cost vs Rating with Play Axis
- Table — Area wise Detail Report
- Slicers — Filter by Area, Price, Rating
